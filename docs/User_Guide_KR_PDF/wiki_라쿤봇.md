# 라쿤봇 (RaccoonBot)

Source: https://github.com/RobomationLAB/RaccoonBot_Guide_KR/wiki/%EB%9D%BC%EC%BF%A4%EB%B4%87

## Overview

RaccoonBot is a compact 4-axis robotic arm designed for educational and research environments — "초소형/초경량의 4축(4자유도) 로봇팔입니다" (an ultra-compact, lightweight 4-axis robotic arm) that operates via Bluetooth 5.0 connectivity.

## Key Features

- Wireless operation with built-in large-capacity battery
- Integration with RobomationLAB development environment
- Teaching & Playback functionality for unplugged learning
- Interchangeable end-effectors for various tasks
- Compatibility with iOS, Android, Scratch, and Python
- Support for peripheral devices like conveyors

## Components

Standard kit includes: robotic arm body, finger gripper, connection cable, Mini Dongle+, USB-C charger, guide stickers, wooden blocks, and manual.

## Safety Notice

The robot weighs approximately 850g and poses risk of damage or injury if dropped from tables.
