# 플러그드 코딩 (Plugged Coding)

Source: https://github.com/RobomationLAB/RaccoonBot_Guide_KR/wiki/%ED%94%8C%EB%9F%AC%EA%B7%B8%EB%93%9C-%EC%BD%94%EB%94%A9

## Main Components

**RobomationLAB** is a Chrome browser-based integrated development environment that abstracts robot hardware into software models, enabling real-time robot control and educational coding experiences.

**Block Composer** provides visual block-based coding for beginners, with automatic JavaScript and Python code generation. Features include drag-and-drop functionality, real-time code execution, and AI-powered feedback.

## Connection Process

1. **Dongle Connection**: Connect a Mini Dongle+ to the PC, which auto-pairs with the RaccoonBot when placed nearby
2. **Connection Verification**: Status can be monitored through browser serial port icons and connection status displays
3. **Robot Addition**: The RaccoonBot blocks are added to the program through a "Select Robot" menu

## Core Features

- Multiple simultaneous robot connections
- Block-based visual programming with syntax error prevention
- Real-time robot feedback and execution
- File management (save, load, initialize)
- Undo/redo functionality and language settings
- Comprehensive example library with code explanations

## Mobile Applications

- **Joystick App** (iOS/Android): Enables manual joint control and interaction with peripheral devices
- **DFU App** (iOS/Android): Provides firmware and feature updates via smartphone
